# Dashboard TB · SINAN NET

Painel interativo de vigilância epidemiológica da tuberculose no Brasil, com dados de notificação do SINAN NET cobrindo 2001 a 2025.

> **Acesso:** [link do dashboard aqui]

---

## O que você encontra no painel

- **Visão geral** — total de casos, taxas de cura, óbito e abandono
- **Mapa** — distribuição geográfica por estado e município
- **Perfil dos pacientes** — sexo, raça/cor, faixa etária, forma clínica
- **Desfechos clínicos** — situação de encerramento, HIV, comorbidades
- **Populações vulneráveis** — privados de liberdade, situação de rua, imigrantes
- **Tendências históricas** — evolução anual e mensal desde 2001
- **Exploração livre** — ferramenta para montar suas próprias análises

---

**Fonte dos dados:** SINAN NET — Ministério da Saúde  
**Cobertura:** notificações de tuberculose, Brasil, 2001–2025
